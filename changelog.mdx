## How to Update

Follow these steps to update your BizNode installation to the latest version.

<Steps>
  <Step title="Backup Your Data" icon="database">
    Create a backup of your memory directory:
    
````bash
cp -r ~/.biznode/memory ~/biznode-backup-$(date +%Y%m%d)
````
  </Step>
  
  <Step title="Download Latest" icon="download">
    <Tabs>
      <Tab title="macOS/Linux" icon="apple">
````bash
curl -L https://biznode.1bz.biz/download/latest/biznode.tar.gz | tar -xz
````
      </Tab>
      <Tab title="Windows" icon="monitor">
        Download from https://biznode.1bz.biz/download and extract.
      </Tab>
    </Tabs>
  </Step>
  
  <Step title="Install and Restart" icon="rocket">
    <CodeGroup tabs="bash">
````bash
sudo ./biznode install --update
biznode restart
````
    </CodeGroup>
    
    Verify with `biznode --version`.
  </Step>
</Steps>

<Callout kind="tip">
After updating, re-verify your node at https://registry.1bz.biz/your-node-id to access premium features like wallet integration.
</Callout>

## Migration Notes

For v1.0.0 users upgrading from v0.x:

<Expandable title="v0.x to v1.0 Migration" default-open="true">
Update your config file from the old YAML format:

<CodeGroup tabs="Before,After">
```yaml
# Before (v0.x)
ai:
  model: qwen2
memory:
  path: ./memory.db
```
```yaml
# After (v1.0+)
ai:
  engine: ollama
  model: qwen2.5-7b
memory:
  engine: qdrant
  path: ~/.biznode/memory
```
</CodeGroup>
</Expandable>
